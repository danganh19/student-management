# Student-Management-BE

Backend for Student Management System

**Cài đặt GORM**
> go get -u github.com/jinzhu/gorm

**Biến môi trường**
* API_HOST_PORT = 0.0.0.0:5000
* DB_HOST = postgres
* DB_PORT = 5432
* DB_NAME = student-management
* DB_PASSWORD = 123456


**Code**
* 404 : Not Found
* 405 : Method not support
* 666 : Null Value
* 777 : Existed
